# rotary-encoder-door-lock (다이얼 도어락)

## 수상
2024 캡스톤디자인 장려상

## 프로젝트 개요
- Standalone Embedded System
- 다이얼(로터리 엔코더) + RFID 2중 인증
- 네트워크 없이 동작하는 접근 제어 시스템

## 시스템 동작 흐름
1. 다이얼 또는 RFID 인증
2. 입력 이벤트 처리
3. 인증 결과 판단
4. LCD UI 및 서보모터 제어
5. 자동 잠금

## 주요 기능
- 다이얼 기반 비밀번호 입력 (채터링 방지)
- RFID UID 인증
- 비밀번호 변경 모드
- 성공/실패 UI 피드백
- 자동 잠금

## 내가 맡은 역할 (팀장)
- 전체 일정 관리 및 기획
- 시스템 구조 설계
- 내부 로직 및 전체 코드 구현
- 입력 처리, 인증, UI, 서보 제어 담당

## 기술적 고민 & 해결
- Servo-LCD 충돌 → PWM 직접 구현
- 엔코더 채터링 → 상태 관리
- RFID 블로킹 → 논블로킹 처리

## 한계점 & 확장 아이디어
- 서버 연동 없음
- 인증 정보 로컬 저장
→ 서버/클라우드 기반 인증 구조로 확장 가능
